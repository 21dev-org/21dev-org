主網（Mainnet）是比特幣的正式生產網路，所有真實的比特幣交易都在此進行。與測試網路不同，主網的比特幣具有實際經濟價值。

## 主網特性

| 特性       | 說明              |
| ---------- | ----------------- |
| 啟動時間   | 2009 年 1 月 3 日 |
| 區塊時間   | 約 10 分鐘        |
| 最大供應量 | 2100 萬 BTC       |
| 難度調整   | 每 2016 區塊      |

## 地址前綴

## 主網與測試網差異

```
主網：
- 真實價值的比特幣
- 全球礦工參與
- 最高安全性
- 不可逆的交易

測試網：
- 無價值的測試幣
- 較低算力
- 可能重置
- 用於開發測試
```

## 連接主網

運行 Bitcoin Core 時，默認連接主網。需要特別指定參數才能連接測試網。
