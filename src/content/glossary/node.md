節點是運行比特幣軟體並連接到比特幣網路的電腦。全節點儲存完整的區塊鏈副本並獨立驗證所有交易和區塊。

## 節點類型

| 類型              | 說明                   | 儲存需求 |
| ----------------- | ---------------------- | -------- |
| **全節點**        | 完整驗證，儲存所有區塊 | ~600 GB+ |
| **修剪節點**      | 完整驗證，刪除舊區塊   | ~10 GB   |
| **輕節點（SPV）** | 只下載區塊頭           | ~50 MB   |
| **礦工節點**      | 全節點 + 挖礦功能      | ~600 GB+ |

## 運行全節點的好處

- **隱私保護**：不需依賴第三方查詢餘額
- **安全驗證**：獨立驗證所有交易
- **支持網路**：增強比特幣網路的去中心化
- **即時廣播**：直接向網路發送交易

## 硬體需求

運行 Bitcoin Core 全節點的最低需求：

- 2 GB RAM
- 600+ GB 硬碟空間（SSD 推薦）
- 穩定的網路連線
- 每月約 200 GB 流量
