減半（Halving）是比特幣協議中預設的機制，每產出 210,000 個區塊（約每 4 年），區塊獎勵會減少一半。這確保了比特幣的通貨緊縮特性和 2,100 萬枚的總量上限。

## 減半歷史

| 次數   | 日期       | 區塊高度 | 獎勵變化         |
| ------ | ---------- | -------- | ---------------- |
| 創世   | 2009-01-03 | 0        | 50 BTC           |
| 第一次 | 2012-11-28 | 210,000  | 50 → 25 BTC      |
| 第二次 | 2016-07-09 | 420,000  | 25 → 12.5 BTC    |
| 第三次 | 2020-05-11 | 630,000  | 12.5 → 6.25 BTC  |
| 第四次 | 2024-04-20 | 840,000  | 6.25 → 3.125 BTC |

## 經濟意義

- **供應衝擊**：新幣發行量突然減少
- **礦工收入**：短期收入減半，需依賴手續費
- **稀缺性**：強化比特幣的數位黃金敘事
- **長期規劃**：可預測的貨幣政策
