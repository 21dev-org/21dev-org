工作量證明（Proof of Work, PoW）是比特幣的共識機制，要求礦工消耗計算資源來證明其對網路的貢獻，以獲得創建新區塊的權利。

## 運作原理

1. 礦工收集待確認交易
2. 反覆嘗試不同 Nonce 計算區塊雜湊
3. 找到小於目標值的雜湊
4. 廣播新區塊並獲得獎勵

## 設計目的

- **成本**：創建區塊需要消耗真實資源（電力、硬體）
- **驗證**：驗證工作量只需一次雜湊計算
- **安全**：攻擊網路需要超過 50% 算力

## 能源消耗

PoW 被批評消耗大量能源，但支持者認為：

- 提供了無與倫比的安全性
- 激勵使用低成本和可再生能源
- 是去中心化的必要代價

## 與 PoS 比較

| 特性     | PoW    | PoS      |
| -------- | ------ | -------- |
| 資源     | 計算力 | 質押代幣 |
| 能耗     | 高     | 低       |
| 去中心化 | 較高   | 較低     |
