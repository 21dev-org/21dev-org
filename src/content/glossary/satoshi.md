聰（Satoshi，簡稱 sat 或 sats）是比特幣的最小單位，以比特幣創造者中本聰（Satoshi Nakamoto）的名字命名。

## 換算關係

```
1 BTC = 100,000,000 sats（一億聰）
1 sat = 0.00000001 BTC
```

## 常見單位換算

| 單位           | 聰數量    | BTC        |
| -------------- | --------- | ---------- |
| 1 sat          | 1         | 0.00000001 |
| 1,000 sats     | 1,000     | 0.00001    |
| 100,000 sats   | 100,000   | 0.001      |
| 1,000,000 sats | 1,000,000 | 0.01       |

## 為什麼使用聰？

隨著比特幣價格上漲，以 BTC 計價日常交易變得不便。使用聰作為單位更直觀，例如一杯咖啡可能是 5,000 sats，而不是 0.00005 BTC。

## 符號

常見的聰符號包括「丰」（U+4E30）和「s」前綴，但目前尚無官方標準符號。
