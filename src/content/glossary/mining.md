挖礦是比特幣網路中驗證交易並將其打包成區塊的過程。礦工透過消耗計算資源來競爭創建新區塊，成功者獲得區塊獎勵和交易手續費。

## 挖礦過程

1. 收集待確認交易組成候選區塊
2. 不斷嘗試不同的 Nonce 值
3. 計算區塊頭的 SHA-256 雜湊
4. 檢查雜湊是否小於目標難度
5. 找到有效雜湊後廣播區塊

## 區塊獎勵

| 時期      | 區塊高度        | 獎勵      |
| --------- | --------------- | --------- |
| 2009-2012 | 0-209,999       | 50 BTC    |
| 2012-2016 | 210,000-419,999 | 25 BTC    |
| 2016-2020 | 420,000-629,999 | 12.5 BTC  |
| 2020-2024 | 630,000-839,999 | 6.25 BTC  |
| 2024-2028 | 840,000+        | 3.125 BTC |

## 挖礦硬體演進

- **CPU**：2009-2010，普通電腦
- **GPU**：2010-2013，顯示卡
- **FPGA**：2011-2013，可程式化晶片
- **ASIC**：2013 至今，專用晶片

## 礦池

個人礦工算力有限，礦池讓多人合作挖礦，按貢獻比例分配獎勵，使收益更穩定。
