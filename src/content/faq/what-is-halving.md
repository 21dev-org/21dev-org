**減半機制**

每產生 210,000 個區塊（約 4 年），礦工的區塊獎勵減少一半：

| 次數 | 年份 | 區塊獎勵   |
| ---- | ---- | ---------- |
| 0    | 2009 | 50 BTC     |
| 1    | 2012 | 25 BTC     |
| 2    | 2016 | 12.5 BTC   |
| 3    | 2020 | 6.25 BTC   |
| 4    | 2024 | 3.125 BTC  |
| 5    | 2028 | 1.5625 BTC |

**為什麼要減半？**

1. **控制通膨**：確保總量不超過 2100 萬
2. **模擬稀缺資源**：類似黃金開採越來越難
3. **公平發行**：早期參與者有優勢但不壟斷

**減半對價格的影響**

歷史上，減半後 12-18 個月內價格都有顯著上漲：

- 2012 減半後：$12 → $1,000+
- 2016 減半後：$650 → $20,000
- 2020 減半後：$8,500 → $69,000

但過去表現不代表未來，請勿視為投資建議。

**礦工怎麼辦？**

減半後收入減少，但：

- 歷史上幣價上漲補償
- 效率低的礦工退出
- 交易手續費佔比增加
