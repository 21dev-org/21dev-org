**簡單解釋**

挖礦是比特幣網路的記帳過程：

1. 礦工收集待處理交易
2. 嘗試解決數學難題（找到有效雜湊）
3. 第一個解出的獲得記帳權
4. 獲得區塊獎勵 + 交易手續費

**為什麼叫「挖礦」？**

類似開採黃金：

- 需要投入資源（設備、電力）
- 有競爭性
- 產出逐漸減少
- 創造新的貨幣供應

**挖礦設備演進**

| 時期  | 設備 | 效率 |
| ----- | ---- | ---- |
| 2009  | CPU  | 極低 |
| 2010  | GPU  | 低   |
| 2013  | FPGA | 中   |
| 2014+ | ASIC | 高   |

**現在還能挖礦嗎？**

個人挖礦幾乎不可能獲利：

- 需要專業 ASIC 礦機
- 電費成本是關鍵
- 通常加入礦池分享收益

**挖礦的意義**

- 保護網路安全
- 公平發行新幣
- 去中心化的共識機制
- 將電力轉換為可信的帳本
