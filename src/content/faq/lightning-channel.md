**開通道的基本步驟**

1. **準備資金**
   - 鏈上需要有比特幣
   - 預留足夠的手續費

2. **選擇對象**
   - 與知名節點開通道
   - 或使用 LSP（閃電服務商）

3. **發起開通道交易**
   - 這是一筆鏈上交易
   - 需要等待確認（通常 3 個）

4. **通道就緒**
   - 可以開始收發閃電支付

**使用錢包開通道**

對新手來說，最簡單的方式是使用自動管理通道的錢包：

| 錢包           | 類型       | 難度 |
| -------------- | ---------- | ---- |
| Phoenix        | 自動開通道 | 簡單 |
| Breez          | 自動開通道 | 簡單 |
| Zeus           | 可自選節點 | 中等 |
| Core Lightning | 完全控制   | 進階 |

**通道容量**

- 你放入多少 BTC，就有多少發送容量
- 接收容量需要對方或透過收款獲得
- 雙向資金通道可以雙向支付

**費用考量**

- 開通道：鏈上手續費
- 使用通道：極低的路由費
- 關通道：鏈上手續費

建議在鏈上手續費低時開通道，然後長期使用。
