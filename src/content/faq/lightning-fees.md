**閃電網路費用結構**

閃電網路的費用遠低於鏈上交易：

| 費用類型 | 金額      | 說明             |
| -------- | --------- | ---------------- |
| 基本費   | 0-1 sat   | 每筆交易固定費用 |
| 比例費   | 0.01-0.1% | 按金額比例收取   |

**實際案例**

發送 10,000 sats（約 $3 USD）：

- 鏈上交易：可能 $1-5 手續費
- 閃電網路：約 1-10 sats（< $0.01）

**費用由誰收取？**

路由節點收取費用：

```
發送者 → 節點A → 節點B → 節點C → 接收者
         ↓        ↓        ↓
       收費     收費     收費
```

每個中間節點設定自己的費率。

**如何降低費用**

1. **直接通道**：與常用對象開直接通道
2. **選擇好的路徑**：錢包會自動優化
3. **使用 LSP**：服務商通常費率較低

**與鏈上比較**

| 特性     | 鏈上     | 閃電網路   |
| -------- | -------- | ---------- |
| 費用基準 | 交易大小 | 交易金額   |
| 小額支付 | 不划算   | 非常划算   |
| 大額支付 | 費用固定 | 按比例增加 |

閃電網路讓微支付成為可能——即使是 1 sat 的交易也經濟可行。
