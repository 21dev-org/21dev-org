**節點是什麼？**

節點是運行比特幣軟體的電腦，它：

- 儲存完整的區塊鏈（約 600 GB）
- 獨立驗證所有交易和區塊
- 與其他節點同步資訊
- 廣播和接收交易

**節點類型**

| 類型     | 儲存 | 驗證 | 用途       |
| -------- | ---- | ---- | ---------- |
| 全節點   | 完整 | 完整 | 最高安全性 |
| 修剪節點 | 部分 | 完整 | 節省空間   |
| 輕節點   | 無   | 部分 | 手機錢包   |

**為什麼要運行節點？**

1. **隱私**：不需依賴第三方查詢餘額
2. **安全**：自己驗證，不信任他人
3. **支持網路**：增加去中心化程度
4. **主權**：真正掌控自己的比特幣

**需要什麼？**

最低需求：

- 電腦（樹莓派也可以）
- 1 TB 硬碟
- 穩定網路
- Bitcoin Core 軟體

**結論**

普通用戶不一定要運行節點，但如果你：

- 持有較多比特幣
- 重視隱私
- 想支持網路

運行自己的節點是值得的投資。
