比特幣錢包是管理你的私鑰和與比特幣網路互動的工具。錢包不儲存比特幣本身，而是儲存用於控制比特幣的密鑰。

## 錢包類型

### 熱錢包（連網）

- **手機錢包**：方便日常使用，如 Muun、Phoenix、BlueWallet
- **桌面錢包**：功能更完整，如 Sparrow、Electrum
- 適合：小額日常支付

### 冷錢包（離線）

- **硬體錢包**：專用安全設備，如 Coldcard、Ledger、Trezor
- **紙錢包**：印刷的私鑰（不建議新手使用）
- 適合：長期儲存大額資金

## 選擇建議

| 用途     | 推薦類型 | 範例               |
| -------- | -------- | ------------------ |
| 日常小額 | 手機錢包 | Muun, Phoenix      |
| 中等金額 | 桌面錢包 | Sparrow, Electrum  |
| 長期儲存 | 硬體錢包 | Coldcard, Ledger   |
| 大額資金 | 多重簽名 | Sparrow + 硬體錢包 |

**記住**：選擇能讓你控制私鑰的錢包！
